**Design System for Budgetr – Inspired by luma.com**

**Philosophy:**
- Quiet confidence – the UI should feel calm, not busy
- Content-first – data is the hero, not decorations
- Subtle delight – micro-interactions that feel premium
- Intentional whitespace – let elements breathe

**Colour Palette:**

Background:
- Primary: #FAFAFA (light) / #0A0A0A (dark)
- Secondary: #FFFFFF (light) / #141414 (dark)
- Tertiary: #F5F5F5 (light) / #1C1C1C (dark)

Text:
- Primary: #171717 (light) / #EDEDED (dark)
- Secondary: #525252 (light) / #A3A3A3 (dark)
- Muted: #A3A3A3 (light) / #525252 (dark)

Accent:
- Primary: #18181B (light) / #FAFAFA (dark) – for buttons, active states
- Success: #22C55E
- Warning: #F59E0B
- Danger: #EF4444

Progress bars:
- Track: #E5E5E5 (light) / #262626 (dark)
- Fill: Linear gradient subtle or solid accent colour

**Typography:**

- Font family: Inter or Geist Sans (variable)
- Headings: 
  - H1: 36px / 600 weight / -0.02em tracking
  - H2: 24px / 600 weight / -0.01em tracking
  - H3: 18px / 500 weight
- Body: 15px / 400 weight / 1.6 line height
- Small/Caption: 13px / 400 weight / text-secondary colour
- Monospace numbers: Use tabular-nums for financial figures

**Spacing:**

- Base unit: 4px
- Component padding: 16px – 24px
- Section gaps: 32px – 48px
- Card padding: 24px
- Max content width: 720px (centred, single column)

**Components:**

Cards:
- Background: secondary bg colour
- Border: 1px solid rgba(0,0,0,0.05) (light) / rgba(255,255,255,0.05) (dark)
- Border radius: 12px
- Shadow: none or very subtle (0 1px 2px rgba(0,0,0,0.04))
- Hover: subtle lift or border change

Buttons:
- Primary: solid black/white, 10px 20px padding, 8px radius, 500 weight
- Secondary: ghost style with subtle border
- Hover: slight opacity change or background shift
- Transition: all 150ms ease

Progress Bars:
- Height: 8px
- Border radius: 9999px (pill)
- Animated fill on load (200ms ease-out)
- Optional: gradient fill for visual interest

Inputs:
- Clean border (1px)
- 12px padding
- 8px radius
- Focus: ring with accent colour
- No inner shadows

**Animations (Framer Motion):**

- Page transitions: fade + slight Y translate (opacity 0→1, y 10→0, 300ms)
- Cards on load: stagger children by 50ms
- Progress bars: width animation on mount
- Hover states: scale(1.01) or subtle shadow
- Number changes: animate with spring for counting up/down

**Layout:**

- Mobile-first single column
- Desktop: still single column, centred (max 720px)
- Sticky header with minimal nav
- Bottom nav on mobile (4 icons max)
- No sidebars – keep it focused

**Iconography:**

- Use Lucide React icons
- Size: 18-20px for UI, 24px for nav
- Stroke width: 1.5px
- Colour: text-secondary, accent on active

**Micro-interactions:**

- Checkboxes: satisfying tick animation
- Adding transaction: card slides in
- Deleting: fade out + height collapse
- Goal reached: confetti or subtle celebration

**General Rules:**

- Never use pure black (#000) or pure white (#FFF) for backgrounds
- Avoid heavy borders – use subtle shadows or bg colour differences
- Numbers should always use tabular-nums for alignment
- Currency should be formatted consistently ($1,234.56)
- Empty states should be friendly and actionable
- Loading states: skeleton shimmer, not spinners
